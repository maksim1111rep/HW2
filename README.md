# Recipe Manager

Система для управления рецептами - консольное приложение, которое позволяет создавать блюда, добавлять их в "рецепты", масштабировать порции и генерировать список покупок.

## Установка

Склонируйте к себе репозиторий:

```bash
git clone https://github.com/maksim1111rep/HW2
cd HW2
```

Установите зависимости:

```bash
pip install -r requirements.txt
```

## Использование

Тесты запускаются командой

```bash
python3 -m pytest
```

Основной код находится в папке `domain`:

- `Ingredient` - ингредиент рецепта;
- `Recipe` - рецепт;
- `DietaryRecipe` - рецепт с диетической категорией;
- `ShoppingList` - список покупок по рецептам.
